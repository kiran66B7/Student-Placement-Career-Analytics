# 🎓 Student Placement Career Analytics

> A Data Analytics and Machine Learning project that analyzes student placement data and predicts placement status using Python and Scikit-learn.

---

## 🚀 Project Overview

This project explores student academic and placement data to discover insights and build a machine learning model that predicts whether a student will be placed.

---

## ✨ Features

- 📊 Data Cleaning & Preprocessing
- 📈 Exploratory Data Analysis (EDA)
- 📉 Data Visualization
- 🔄 Label Encoding
- 🤖 Random Forest Classification
- 📋 Model Evaluation
- 📌 Feature Analysis

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab
- GitHub

---

## 📂 Dataset Features

- Gender
- SSC Percentage
- HSC Percentage
- Degree Percentage

- Work Experience

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/kiran66B7/Student-Placement-Career-Analytics.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Open the notebook

```text
Student_Placement_Career_Analytics.ipynb
```

4. Run all cells.
